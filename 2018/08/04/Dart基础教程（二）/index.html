内置类型

Dart语言对以下类型的特殊支持：

  • 数字 numbers
  • 字符串 strings
  • 布尔 booleans
  • 列表 lists (也叫数组 arrays)
  • map 集合 maps
  • runes (for expressing Unicode characters in a string)字符（用于在字符串中表示Unicode字符）
  • 符号 symbols

您可以使用常量来初始化任何这些特殊类型的对象，比如：’this is a string’是字符串常量，而ture的是一个布尔值
因为在 Dart 中，每一个变量是指一个对象，即一个类的实例，通常可以使用构造函数初始化变量的对象实例。一些内置的类型都有自己的构造函数。例如，你可以使用 Map() 构造函数来创建一个映射，使用代码，如 new Map()。

Numbers

Dart 数字有两种形式：

int

整数值不大于64位，这个取决于平台。在Dart虚拟机中，值可以从-2的63次方到2的63次方减一。编译成JavaScript的Dart使用的是JavaScript numbers，所以值是-2的63次方到2的63次方减一-2的53次方到2的53次方减一

double

64位（双精度）浮点数，由IEEE 754制定标准指定。
int和double是num的子类型。该数字类型包括基本运算符，如+， - ，/和*，同时也有abs(), ceil(), 和 floor()等其它方法，也支持位运算符，例如 >>。如果num及其子类型没有你要找的方法或内容，dart:math库可能有。

Runes

在Dart 中 runes 是UTF-32字符集的string 对象。
Unicode为世界上所有的字母、数字和符号定义了一个惟一的数值。因为Dart字符串是UTF-16编码单元的序列，所以在字符串中表示32位Unicode值需要特殊的语法。
表达Unicode代码点的通常方法是uXXXX，其中XXXX是一个4位的十六进制值。

Symbols

一个符号对象表示在Dart程序中声明的操作符或标识。你可能不会需要使用这些符号，但是对于那些引用标识符名称的api来说，它们是很重要的，因为时常要改变的是标识符的名称，而不是标识符的符号
要获得标识符的符号，请使用符号文字#，后面跟着标识符
